# Student Management System (Flask)

## Overview
A web-based student management system built using Flask and SQLite...

## Features
- Add student
- Update student
- Delete student
- View all students
- Simple authentication system

## Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS

## How to Run
pip install -r requirements.txt
python app.py

## Future Improvements
- Role-based login
- Search functionality
